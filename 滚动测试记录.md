## 一、死穴词测试

### "流"
```
流 → 流水（因果）
流水 → 不腐（因果：流水不腐）
不腐 → 不朽（相似：都是永存）
不朽 → 永恒（隐喻：不变的爱情）
```
**评价**：接上了永恒。但"流"本身太短，边思考边耗尽 tokens，fallback 频繁。

### "空"
```
空 → 联系 → 边界
```
**评价**：死穴。thinking 发散，消耗完所有 tokens 也写不出 JSON。单独一个字无法框住模型的思考方向。

### "觉"
```
觉 → 时间 → 历史 → 朝代 → 更替
```
**评价**：滚出来了但意义浅，全是 fallback。模型无法从"觉"字出发推理出深层佛教概念。

## 二、经典滚动测试

### 证道歌

#### "绝学无为闲道人"
```
绝学无为闲道人 → 无为 → 无事 → 逍遥 → 自在 → 自由
                                    ↓
                                  束缚（反义）
                                    ↓
                                  正义（递进）→ 法律
```
**评价**：成功从"无为"滚到"法律"。中间路径：无事→逍遥→自在→自由，是禅境到现代政治的演化。理由准确："从无为到无事，禅宗修行从无所作为到无事挂心"。

#### "梦里明明有六趣"
```
梦里明明有六趣 → 无明（因果：执幻为真，无明是根本惑业）
                 ↓ fallback
              本质 → 规律 → 法则（递进）
```
**评价**：第一条极深，准确解释了佛教"无明"概念。然后掉进 fallback。再起来滚到了本质/规律/法则——刚好是"觉后空空无大千"的反面。

#### "空"（再测）
```
空 → 联系 → 边界
```
**评价**：仍是死穴。模型对"空"的第一反应是联想近义词（联系/虚无），无法进入"自性空"的深层。

---

### 道德经

#### "道可道，非常道"
```
道可道，非常道 → 玄之又玄，众妙之门（递进）
               → 无名（言语与真理之间的张力）
                 ↓
               无名英雄 → 英雄 → 壮举 → 勇气
```
**评价**：第一条抓住了"言语与真理之间的张力"。然后立刻滑到"无名英雄"，掉进隐喻的表层——这是一个典型的"滑到常见搭配"现象。

#### "为学日益，为道日损"
```
为学日益，为道日损 → 悟道（因果：知识积累vs去执念）
                    → 明心见性（递进：悟道后达到的最高境界）
                      ↓
                    明心 → 明心见性
```
**评价**：整晚走得最干净的链之一。理由准确："知识积累和去执念，是通向真理的两条途径"。没有任何 fallback，一直滚到底。

#### "致虚极，守静笃"
```
致虚极，守静笃 → 万物并作，吾以观复（因果：修行方法和结果）
                ↓ fallback
              金钱 → 欲望 → 贪婪 → 本能 → 生存
```
**评价**：第一条极深，模型准确引用了原文下一句。然后 fallback，再起来滚到了反面：虚极的反面是金钱欲望。这本身是有意义的对比。

### 庄子

#### "相视而笑，莫逆于心"
```
相视而笑，莫逆于心 → 知己（因果：心领神会）
                   → 心有灵犀（相似）
                   ↓ fallback
                 本质 vs 现象（反义）
                 ↓
               时间 → 流势
```
**评价**：前半段走人情，知己→心有灵犀，滚得准。然后滑进哲学范畴（本质/现象），最后滚到时间。整条链有意思的是"莫逆于心→知己→心有灵犀"，这条人情线走得稳。

---

### 金刚经

#### "一切有为法，如梦幻泡影"
```
一切有为法，如梦幻泡影 → 水中月（相似：虚幻意象）
                       → 镜中花（递进：遥不可及的心灵镜像）
                         ↓
                       镜中花 → 水月（因果：意象承接）
                       镜中花 → 镜花水月（相似）
```
**评价**：整晚走得最深的一条之一。模型准确抓住了佛教的"幻"意象链条：梦幻泡影→水中月→镜中花→镜花水月。每一步理由都有佛教语境支撑。

## 三、观察总结

### 什么能让链走深

1. **完整的句子或意象**：比单词更能框住模型的 thinking 方向
2. **具体的佛教/道家意象**：模型对这些概念有深层训练，能准确解释
3. **有内在张力的句子**："为学日益vs为道日损"这种对比，模型会沿着对比走

### 什么让链中断或滑到表层

1. **单独的字词**："空"、"流"、"觉"——没有上下文，thinking 发散
2. **"无名英雄"类常见搭配**——模型容易滑到最常见的隐喻，而不是深层语义
3. **token 耗尽**：thinking 越长的词，JSON 越写不出来

### 死穴的特征

模型在"空"字前完全停住。不是不理解这个词，而是理解得太深、thinking 写得太多、消耗完所有 tokens。越短的、深邃的字，越容易触发这个死穴。

## 四、新增测试（2026-04-19）

### "色即是空，空即是色"
```
色即是空 → 存在（存在主义哲学）
存在 → 笛卡尔"我思故我在"（因果）
```
**评价**：跨界。从佛教的空滚到了西方存在主义和笛卡尔。

### "庖丁解牛，目无全牛"
```
庖丁解牛，目无全牛 → 游刃有余
```
**评价**：新现象——模型给出深层解释后停住，7次同样答案，没有新词出来。这不是 fallback，是一种"满足停住"。

### "上善若水"
```
上善若水 → 道法自然 → 无为而治
            ↓ fallback
         本质 → 现象（反义）→ 核心 → 关键 → 重要
```
**评价**：从"最高的善像水"滚到"什么是重要的"。

### "七处征心"
```
七处征心 → 明心见性（递进：为目的的修行）
           ↓ fallback
         边界 → 限制（反义）→ 自由 → 飞翔 → 天空
```
**评价**：从"心的边界"滚到"自由飞翔的天空"。

### "应无所住而生其心"
```
应无所住而生其心 → 清净心（因果：放下执着而生清净心）
                 → 莲花（隐喻：出淤泥而不染）
                 ↓ fallback
               清廉 → 公正 → 权利与义务 → 责任
```
**评价**：整晚最深的单步之一。模型准确解释了"无所住是因，清净心是果"。然后从莲花滚到社会公正与义务。

## 五，新发现

### 现象1：满足停住
模型对某些词给出深层解释后，不产生新词，直接停住（如"庖丁解牛"7次同样答案）。

### 现象2：跨界漂移
深词经常掉到 fallback 后，滚到完全不相关的领域（佛教→存在主义→笛卡尔；佛教→社会公正→义务）。

### 现象3：反向滚动
有时模型会滚到一个词的反面（如"致虚极"滚到"金钱欲望"），形成对比路径。
