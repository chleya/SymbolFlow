## 一、死穴词测试

### "流"
```
流 → 流水（因果）
流水 → 不腐（因果：流水不腐）
不腐 → 不朽（相似：都是永存）
不朽 → 永恒（隐喻：不变的爱情）
```
**评价**：接上了永恒。但"流"本身太短，边思考边耗尽 tokens，fallback 频繁。

### "空"
```
空 → 联系 → 边界
```
**评价**：死穴。thinking 发散，消耗完所有 tokens 也写不出 JSON。单独一个字无法框住模型的思考方向。

### "觉"
```
觉 → 时间 → 历史 → 朝代 → 更替
```
**评价**：滚出来了但意义浅，全是 fallback。模型无法从"觉"字出发推理出深层佛教概念。

## 二、经典滚动测试

### 证道歌

#### "绝学无为闲道人"
```
绝学无为闲道人 → 无为 → 无事 → 逍遥 → 自在 → 自由
                                    ↓
                                  束缚（反义）
                                    ↓
                                  正义（递进）→ 法律
```
**评价**：成功从"无为"滚到"法律"。中间路径：无事→逍遥→自在→自由，是禅境到现代政治的演化。理由准确："从无为到无事，禅宗修行从无所作为到无事挂心"。

#### "梦里明明有六趣"
```
梦里明明有六趣 → 无明（因果：执幻为真，无明是根本惑业）
                 ↓ fallback
              本质 → 规律 → 法则（递进）
```
**评价**：第一条极深，准确解释了佛教"无明"概念。然后掉进 fallback。再起来滚到了本质/规律/法则——刚好是"觉后空空无大千"的反面。

#### "空"（再测）
```
空 → 联系 → 边界
```
**评价**：仍是死穴。模型对"空"的第一反应是联想近义词（联系/虚无），无法进入"自性空"的深层。

### 道德经

#### "道可道，非常道"
```
道可道，非常道 → 玄之又玄，众妙之门（递进）
               → 无名（言语与真理之间的张力）
                 ↓
               无名英雄 → 英雄 → 壮举 → 勇气
```
**评价**：第一条抓住了"言语与真理之间的张力"。然后立刻滑到"无名英雄"，掉进隐喻的表层——这是一个典型的"滑到常见搭配"现象。

#### "为学日益，为道日损"
```
为学日益，为道日损 → 悟道（因果：知识积累vs去执念）
                    → 明心见性（递进：悟道后达到的最高境界）
                      ↓
                    明心 → 明心见性
```
**评价**：整晚走得最干净的链之一。理由准确："知识积累和去执念，是通向真理的两条途径"。没有任何 fallback，一直滚到底。

#### "致虚极，守静笃"
```
致虚极，守静笃 → 万物并作，吾以观复（因果：修行方法和结果）
                ↓ fallback
              金钱 → 欲望 → 贪婪 → 本能 → 生存
```
**评价**：第一条极深，模型准确引用了原文下一句。然后 fallback，再起来滚到了反面：虚极的反面是金钱欲望。这本身是有意义的对比。

### 庄子

#### "相视而笑，莫逆于心"
```
相视而笑，莫逆于心 → 知己（因果：心领神会）
                   → 心有灵犀（相似）
                   ↓ fallback
                 本质 vs 现象（反义）
                 ↓
               时间 → 流势
```
**评价**：前半段走人情，知己→心有灵犀，滚得准。然后滑进哲学范畴（本质/现象），最后滚到时间。整条链有意思的是"莫逆于心→知己→心有灵犀"，这条人情线走得稳。

### 金刚经

#### "一切有为法，如梦幻泡影"
```
一切有为法，如梦幻泡影 → 水中月（相似：虚幻意象）
                       → 镜中花（递进：遥不可及的心灵镜像）
                         ↓
                       镜中花 → 水月（因果：意象承接）
                       镜中花 → 镜花水月（相似）
```
**评价**：整晚走得最深的一条之一。模型准确抓住了佛教的"幻"意象链条：梦幻泡影→水中月→镜中花→镜花水月。每一步理由都有佛教语境支撑。

## 三、观察总结

### 什么能让链走深

1. **完整的句子或意象**：比单词更能框住模型的 thinking 方向
2. **具体的佛教/道家意象**：模型对这些概念有深层训练，能准确解释
3. **有内在张力的句子**："为学日益vs为道日损"这种对比，模型会沿着对比走

### 什么让链中断或滑到表层

1. **单独的字词**："空"、"流"、"觉"——没有上下文，thinking 发散
2. **"无名英雄"类常见搭配**——模型容易滑到最常见的隐喻，而不是深层语义
3. **token 耗尽**：thinking 越长的词，JSON 越写不出来

### 死穴的特征

模型在"空"字前完全停住。不是不理解这个词，而是理解得太深、thinking 写得太多、消耗完所有 tokens。越短的、深邃的字，越容易触发这个死穴。

## 五，新发现

### 现象1：满足停住
模型对某些词给出深层解释后，不产生新词，直接停住（如"庖丁解牛"7次同样答案）。

### 现象2：跨界漂移
深词经常掉到 fallback 后，滚到完全不相关的领域（佛教→存在主义→笛卡尔；佛教→社会公正→义务）。

### 现象3：反向滚动
有时模型会滚到一个词的反面（如"致虚极"滚到"金钱欲望"），形成对比路径。

## 六、新增测试记录（第二轮）

### "不知色身，外洎山河虚空大地，皆是妙明真心中物"
```
皆是妙明真心中物 → 色身（起点亦是终点，一切不出自心）
                 → 法身（递进：色身vs法身）
                   → 报身 → 圆满 → 完美 → 极致
```
**评价**：第一步极准——"色身为起点亦是终点，揭示一切法不出自心"。然后沿着佛教三身走，走到了完美/极致。有意思的是：从"不出自心"滚到了"极致"——越追求心，越走到极端。

### "凡所有相，皆是虚妄"
```
凡所有相，皆是虚妄 → 存在 vs 本质（萨特：存在先于本质）
                   → 本质 → 实质（相似）
                   → 本质 → 核心（递进）
                   → 本质 → 规律（因果：本质决定规律）
```
**评价**：从"一切皆虚"滚到了"规律"。表面是虚，背后有律。老子说大道废有仁义——越是表层虚的，深层越有实在的规律。

### "祸兮福所倚，福兮祸所伏"
```
祸兮福所倚 ↔ 塞翁失马（相似）
```
**评价**：模型认为这两句话是等价关系，所以在它们之间循环。这是"等价满足停住"。

### "天地与我并生，而万物与我为一"
```
天地与我并生 → 天人合一（递进）
天人合一 → 道法自然（因果：道法自然是天人合一的哲学基础）
道法自然 → 天道（因果：天道是宇宙因果规律的体现）
天道 → 因果（因果：天道本身就是因果规律）
```
**评价**：整晚走得最干净的链之一，没有断裂。从庄子的"天人合一"滚到了"因果"，然后在天道和因果之间循环，因为两者互为定义。

### "照见五蕴皆空"
```
照见五蕴皆空 → 度脱苦难（递进）
五蕴皆空 → 四大皆空（相似）
四大皆空 → 看破红尘 → 出家修行 → 遁入空门 → 因果报应
```
**评价**：从"五蕴皆空"滚到了"因果报应"。越空的东西，最后滚到了最实的因果律。

### "为学日益，为道日损，损之又损"
```
损之又损 → 无为
无为 → 返璞归真（隐喻）
返璞归真 → 纯真（因果）
纯真 → 天真（相似）
纯真 → 童年（递进）
```
**评价**：在"返璞归真"和"纯真"之间循环，因为它们互相定义——返璞归真的目标就是纯真，纯真的状态就是返璞归真。

### "知天之所为，知人之所为"
```
知天之所为，知人之所为 → 天人合一（递进：知天道而行自然之事）
天人合一 → 道法自然（因果）
        ↓ fallback
      因果
      ↓
    存在 ←→ 虚无（反义）
```
**评价**：从"天人合一"滚到了"存在 vs 虚无"的对立，然后在两者之间循环。这是佛教/道家最深的对立——有与无。

### "天地一指也，万物一马也"
```
天地一指也，万物一马也 → 无（消解万物差异）
                   ↓ fallback
                 时间 → 流逝
```
**评价**：从"齐物论"（万物本无差别）滚到了"流逝消亡"——万物的差别被消解了，最后归于时间。

### "出生入死"
```
出生入死 → 舍生忘死（相似）
          贪生怕死 vs 视死如归（反义）
          视死如归 → 英勇（因果：无私精神的高贵）
            → 勇敢 → 胆量 → 勇气 → 无畏
```
**评价**：从"出生入死"滚到了"无畏"。模型在找一条从生死观到精神境界的路。

### "大成若缺，其用不弊"
```
大成若缺 → 圆满 → 完美 → 极致
极致 → 边界 → 限制
```
**评价**：完美到了极点就到了边界。大成若缺——最圆满的看起来有缺陷，用起来却不弊。

### "纵迷一切种因，亦得成菩提"
```
纵迷一切种因，亦得成菩提 → 因缘际会（因果：种因得果）
                         → 觉悟（因果：迷惑中成就觉悟）
觉悟 → 觉醒（相似）
觉醒 → 存在（递进：存在需要意识来确认）
```
**评价**：最后一步极深——"没有意识，存在没有意义"。迷惑中也能觉悟，然后觉悟到存在需要意识来确认。

### "物物而不物于物"
```
物物而不物于物 → 逍遥（递进：驾驭外物而不被外物束缚的最高境界）
               → 齐物（递进：万物齐一，物我两忘）
```
**评价**：从"驾驭外物而不被外物束缚"滚到了"齐物"——这是庄子里程碑式的一句话。

### "大道不称，大辩不言"
```
大道不称 → 无为（因果：真正的大道无需命名和称谓）
无为 → 本质 → 规律 → 规则 → 定律 → 法则
```
**评价**：从"无为"滚到了"法则"。道不称、辩不言、仁不仁——真正的大道最后滚成了法则。

### "少则得，多则惑"
```
少则得 → 专注（递进）
        → 简约（隐喻）
简约 → 极简（递进）
极简 → 断舍离（递进：从理念到行动的深化）
```
**评价**：从老子的"少则得"滚到了日本极简主义的"断舍离"。东方哲学和现代极简主义的接上了。

### "真空不空，妙有非有"
```
真空不空，妙有非有 → 无极（递进：宇宙本体从虚无创生）
                  ↓ fallback
                时间 → 流势
```
**评价**：第一条极深——"真空妙有"揭示有与无的辩证关系，联想到道家"无极"，佛道宇宙论互补。然后滚到了"时间流逝"。

### "真空不空，妙有非有"（完整句）
```
真空不空，妙有非有 → 无极
                  → 因果循环
                  → 循环
```
**评价**：从"真空"滚到了"循环"。

### "相忘于道术"
```
相忘于道术 → 逍遥游（相似：精神绝对自由的境界）
            → 坐忘（相似：物我两忘的精神境界）
逍遥 → 自在（递进）
```
**评价**：整条链都在庄子精神自由的范畴里走，没有断裂。

### "狂心顿歇，歇即菩提"
```
狂心顿歇，歇即菩提 → 放下（递进：放下是觉悟的途径，歇是放下的当下呈现）
                   ↓ fallback
                 因果 → 循环 → 回归 → 重返 → 故里
```
**评价**：极深——歇即放下，放下是觉悟的途径。然后从"放下"滚到了"乡愁"——停下来的那一刻就是回到本源。这老子说的"归根复命"。

### "归根复命"
```
归根复命 → 落叶归根（隐喻：万物归于本原）
         → 返本还原（因果）
         → 周而复始（因果）
         → 循环 → 周期 → 节律 → 节奏
```
**评价**：整晚最深的链之一。从"归根复命"滚到了"节奏"——落叶归根、周而复始——这是宇宙的节奏。万物的周期性的，最后滚到了最日常的节奏。

## 七、总体观察

### 最深的几条链

1. **归根复命 → 落叶归根 → 周而复始 → 循环 → 周期 → 节律 → 节奏**
2. **为学日益，为道日损 → 悟道 → 明心见性 → 明心**
3. **天地与我并生 → 天人合一 → 道法自然 → 天道 → 因果**
4. **狂心顿歇，歇即菩提 → 放下 → 回归 → 故里**

### 规律总结

1. **越深的句子，越容易触发 token 耗尽型死穴**：thinking 写得太多，JSON 写不出来
2. **fallback 后最常滚到的领域**：时间/流逝/循环/因果/规律
3. **跨界漂移的规律**：从宗教/哲学掉到 fallback 后，经常滚到"时间-存在-因果-规律"这条线
4. **最不容易断裂的链**：有具体意象的（落叶/流水/莲花/逍遥），而不是抽象概念

## 八、新增测试记录（第三轮）

### "致虚极，守静笃。万物并作，吾以观复"
*测试时间：2026-04-19 傍晚*

```
致虚极，守静笃。万物并作，吾以观复 → 归根（因果：原文紧接着"夫物芸芸，各复归其根"，观复的目的是观察万物回归根本）
               → 本质（fallback）
               → 规律 → 规则 → 选择 → 决定 → 核心
```
**评价**：第一条极深——模型准确引用原文"各复归其根"，解释力极强。后续滚到本质→规律→规则→选择→决定→核心，走的是哲学上层建筑到个人决策的路径。

### "知止乎不知，至矣"
```
知止乎不知，至矣 → 时间（fallback）
                 → 联系 → 知足 → 关系 → 常乐 → 联想词 → 边界 → 本质
```
**评价**：从"知止乎不知"滚到了"知足"。知止和知足——庄子的相对主义在这里变成了人生智慧。知足常乐是中国哲学最常见的应用链。

### "法尚应舍，何况非法"
```
法尚应舍，何况非法 → 联系（fallback）
                  → 虚静 → 关系 → 无为 → 有为 → 因果 → 沟通 → 循环 → 时间
```
**评价**：从"法尚应舍"滚到了"无为"和"有为"的对比——这是《金刚经》的核心。然后滚到因果→沟通→循环→时间。因果链条可以形成闭环这个观察很准确。

### "无智亦无得"
```
无智亦无得 → 交流（fallback via 沟通）
           → 无所得
```
**评价**：只走了一步就停了。《心经》最深的两句之一（另一句是"色即是空"），模型无法深入。死穴。

## 九、下次测试建议

### 为什么句子比单词更容易走深

| 类型 | 示例 | 表现 |
|------|------|------|
| 单词 | 空、流、觉 | thinking 发散，token 耗尽，JSON 写不出 |
| 短句 | 上善若水、相视而笑 | 有意象，能框住 thinking 方向 |
| 长句 | 致虚极，守静笃。万物并作 | 有内在张力，模型会抓住矛盾演化 |

### 深层链的共同特征

1. **有具体意象**：落叶、流水、莲花、逍遥、镜花水月
2. **有内在矛盾**：为学vs为道、损之又损、致虚极vs万物作
3. **有宗教/哲学语境**：佛教、道家概念有深层训练支撑

### 跨界漂移的路径图

```
佛教/道家深层概念
    ↓ fallback
时间/因果/规律/循环
    ↓ 继续滚
存在 vs 虚无
    ↓ 或
现实世界（法律/政治/社会责任）
```

## 十五、第四轮观察：禅宗链的巅峰

"至道无难，唯嫌拣择 → 无住 → 无念" 是整晚最干净的禅宗链：
