# SymbolFlow — 语言关联可视化工具

## 核心理念

> "不是人在思考，不是 LLM 在联想。是某个东西借人和 LLM 的口在流，SymbolFlow 是那个东西流过的痕迹。"

- 种子词 = 给"那个东西"一个起点，不是终点
- LLM 的游走 = "那个东西"在符号空间里迈出的步伐
- 人的判断 = 留住值得留的脚印，让其他的流过去
- **最终产物不是地图，是痕迹——是某个东西曾经流过的证明**

**本质：不是工具，是载体。是人 + LLM 一起接住"那个东西"的地方。**

**目的：让人看到自己语言思维里本来看不到的东西。**

## 核心流程

### 1. 种子输入
- 用户在顶部输入框输入一个"种子"（词/概念/问题）
- 按回车或点击"开始游走"
- 种子作为第一个节点出现在画布中央

### 2. LLM 游走机制
LLM 根据当前节点，生成下一步跳转：
- 当前节点 → LLM 判断"钩到这里有意思" → 下一个节点
- 同时生成"钩子理由"：为什么跳到这里

**提示词示例（发给 LLM 的钩子）：**
```
当前节点: {current_node}
已有的连接: {existing_edges}

请判断从当前节点最值得游走的方向是什么。
返回一个 JSON：
{{
  "next_node": "下一个节点（一个词或短句）",
  "reason": "为什么跳到这里（20字以内）",
  "connection_type": "因果/相似/反义/隐喻/递进"
}}
```

### 3. 可视化
- 使用 D3.js 力导向图
- 节点：词语/概念
- 边：游走路径，带方向箭头
- 边的标签：connection_type（因果/相似/反义/隐喻/递进）
- 最新生成的节点高亮，路径动画扩散

### 4. 用户干预
| 操作 | 效果 |
|------|------|
| 点击节点 | 选中，显示信息面板 |
| 双击节点 | 聚焦：从这个节点重新开始游走 |
| 拖拽节点 | 自由布局 |
| 右键节点 | 菜单：添加子节点 / 固定 / 删除 |
| 右键边 | 删除这条连接 |
| 拖拽添加边 | 从节点A拖到节点B，手动创建连接 |
| 添加节点按钮 | 在选中节点下新建空白子节点，用户输入内容 |
| 固定节点 | 该节点不会被新路径覆盖 |
| 清空 | 重置画布 |

### 5. 暂停/继续
- 点击"暂停"：LLM 停止生成新节点
- 画布保持当前状态，可自由编辑
- 点击"继续"：从最后一个节点继续游走

### 6. 游走日志（核心功能）

每次游走自动完整记录：
```js
log = {
  seed: '自由',
  startTime: '2026-04-17T12:00:00',
  steps: [
    {
      step: 1,
      from: '自由',
      to: '选择',
      reason: '自由常与选择相连，选择带来责任',
      connectionType: '因果',
      timestamp: '2026-04-17T12:00:01',
    },
    {
      step: 2,
      from: '选择',
      to: '责任',
      reason: '每个选择都伴随责任的大小',
      connectionType: '因果',
      timestamp: '2026-04-17T12:00:02',
    },
    // ...
  ],
  totalSteps: 12,
}
```

**历史记录面板（侧边栏）：**
- 每次游走自动存入 LocalStorage
- 显示：种子词 + 时间 + 步数 + 预览缩略图
- 点击可：查看详情 / 回放 / 对比 / 删除

**回放模式：**
- 选择某次历史记录，点击"回放"
- 画布上逐步重现游走过程（每步 0.8s，可调速）
- 节点依次出现，边以动画绘制
- 回放过程中可暂停/加速/减速

**对比模式：**
- 选择同一个种子的多次游走记录
- 并排展示两张图，高亮差异节点
- 统计：重合节点比例、平均路径长度、共同边数

### 7. 导出
- **PNG**：当前画布截图
- **JSON**：完整图数据（节点、边、步骤日志）
- **Markdown**：以树状结构展示思维流
- **完整日志 JSON**：包含每一步的 from/to/reason/timestamp

## 交互细节

### 启动
1. 显示暗色 splash 画面（SymbolFlow logo + 3个动画点）
2. 2秒后淡出，主界面出现
3. 顶部种子输入框自动获得焦点

### 空状态
- 画布中央显示提示文字："输入一个种子词，按回车开始游走"
- 画布背景：深灰 #1a1a2e，带微弱网格

### 游走进行中
- 新节点从当前节点位置向外扩散（d3 力学模拟）
- 边以动画方式绘制（从源到目标的线条生长动画）
- 当前节点边框发光（表示"在这里"）

### 节点信息面板（选中节点时）
- 节点标签（可编辑）
- 钩子理由（该节点从哪跳来的）
- 入度 / 出度
- 操作按钮

### 设置面板（齿轮图标）
- API Base URL
- API Key（密码框）
- 模型选择
- Temperature 滑块（0.5~1.5）
- 保存到 LocalStorage
